Lesson: Strings in Processing

Strings are Objects

A string is a sequence of characters:


String s = "Hello";

Important: a String is an object, not a primitive.

⚠️ String Comparison

Never use == to compare strings.

Misleading case


String a = "hello";
String b = "hello";

println(a == b); // might print true

This works only because the compiler may reuse the same object.

Real case


String a = "hello";
String b = "he" + "llo";

println(a == b); // false

Same text, different objects.

Correct way


println(a.equals(b)); // true

Rule: use .equals()

Immutability

Strings cannot be changed after creation.


String word = "cat";
word = "cut"; // new object

You are not editing the old string — you are creating a new one.

Fixing a typo

This does NOT work:


String typo = "I liek Processing";
typo.replace("liek", "like");
println(typo); // still wrong

Because replace returns a NEW string.

This works:


typo = typo.replace("liek", "like");
println(typo);

Useful Methods


String s = "Processing";

s.length();
s.charAt(0);
s.substring(0,4);

Typography


text("Hello", 100, 100);

Fonts


PFont f = createFont("Arial", 32);
textFont(f);

Alignment


textAlign(CENTER, CENTER);

Flexibility Example

Turn a string into characters:


String word = "Flexibility";
char[] letters = word.toCharArray();

Now each letter can be manipulated independently.
